## 1. 경쟁사 현황

### 1.1 직접 경쟁사 (AI 격려 + 알림 특화)

#### Remindly (remindly.app)
| 항목 | 내용 |
|------|------|
| 플랫폼 | iOS / Android |
| 핵심 기능 | AI가 격려 또는 자책 톤으로 알림 발송, 스트릭(연속 달성) 추적, 커스텀 알림 |
| 차별화 | 알림 톤 선택 가능 (격려/자책/중립), 이모지·컬러 비주얼, 습관 추적 |
| 한계 | 캘린더/일정 관리 기능 없음, 업무용 알림 채널(Teams 등) 미지원, AI 개인화 깊이 낮음 |
| 유사도 vs Cheerly | **중간** — 격려 메시지 컨셉은 동일하나 일정 중심이 아닌 습관/할일 앱 |

#### HiFutureSelf (hifutureself.com)
| 항목 | 내용 |
|------|------|
| 플랫폼 | iOS / Android / Web |
| 핵심 기능 | 미래의 자신에게 메시지 예약 발송, 동기부여 리마인더, 개인 성장 일지 |
| 차별화 | 감성적 스토리텔링, "미래 자아" 컨셉, 장기 목표 동기부여 |
| 한계 | 일정 관리 도구가 아님, 비즈니스 채널 알림 없음, AI 메시지 자동화 미흡 |
| 유사도 vs Cheerly | **낮음** — 감성적 방향은 유사하나 일정 리마인더 용도가 아님 |

#### Rocky.ai (rocky.ai)
| 항목 | 내용 |
|------|------|
| 플랫폼 | Web / Mobile |
| 핵심 기능 | AI 성장 마인드셋 코치, 리더십 역량 개발, 동기부여 푸시 알림 |
| 차별화 | 구조화된 학습 경로(최대 6개 리더십 주제), 팀 단위 코칭 |
| 한계 | 일정 관리 기능 없음, 업무 도구 통합 없음, B2B 전문 도구 |
| 유사도 vs Cheerly | **낮음** — 코칭 플랫폼이며 일정 알림 서비스가 아님 |

## 3. Cheerly만의 차별화 강점

### 3.1 핵심 차별화: "알림 + 격려" 통합

| 경쟁사 | 알림 | AI 격려 | 업무 채널 통합 | 일정 관리 |
|--------|------|---------|--------------|----------|
| Google Calendar | ✅ | ❌ | ❌ | ✅ |
| Remindly | ✅ | ⚠️ 일반적 | ❌ | ❌ |
| Todoist | ✅ | ❌ | ⚠️ 부분 | ✅ |
| **Cheerly** | **✅** | **✅ Claude AI** | **✅ Teams** | **✅** |

### 3.2 상세 차별화 항목

#### ① 일정 컨텍스트 반영 AI 메시지
- **경쟁사**: 일반적인 동기부여 문구("오늘도 화이팅!") 수준
- **Cheerly**: Claude AI가 일정 제목 + 카테고리를 분석하여 "팀 회의를 앞두고 계시군요. 충분히 준비되었습니다. 자신감 있게 임해보세요!" 같은 **상황 맞춤 메시지** 생성

#### ② 업무용 알림 채널 직접 지원
- **경쟁사**: 모두 스마트폰 푸시 알림만 지원
- **Cheerly**: Microsoft Teams Adaptive Card 지원 → **업무 중 Teams에서 직접 알림 수신**
- 향후: KakaoTalk, Slack 확장 예정 (Strategy Pattern으로 쉽게 추가 가능)

#### ③ 폴백 내결함성
- **경쟁사**: AI 서버 다운 시 알림 자체 실패
- **Cheerly**: AI 실패 시 카테고리별 44개 정적 메시지로 즉시 폴백 → **알림은 항상 발송**

#### ④ 개발자 친화적 오픈 아키텍처 (MVP 단계)
- 로컬 SQLite → PostgreSQL 전환 용이 (어댑터 교체만)
- 알림 채널 추가 시 `scheduler.ts` 수정 없이 클래스만 추가
- 환경변수 하나로 AI 모델 교체 가능 (`ANTHROPIC_MODEL`)

#### ⑤ 무설치 로컬 배포 가능
- SQLite 기반 → DB 서버 없이 단독 실행
- `npm run dev` + 환경변수 3개만으로 즉시 사용 가능

## 4. 시장 기회 분석

### 4.1 타겟 시장 규모

| 세그먼트 | 규모 | Cheerly 적합도 |
|---------|------|--------------|
| 직장인 회의 리마인더 | 글로벌 화이트칼라 약 1.2B명 | ★★★★★ |
| 번아웃 예방/동기부여 니즈 | 직장인의 52% 번아웃 경험(Gallup 2024) | ★★★★☆ |
| Teams 사용 기업 | Microsoft 365 MAU 3억 2천만명 | ★★★★★ |
| 한국 직장인 | 약 2,700만명, 높은 업무 스트레스 | ★★★★★ |

### 4.2 현재 미충족 니즈 (Pain Points)

1. **"알림은 오지만 준비할 마음이 안 생긴다"** — 기계적 알림의 한계
2. **"Teams를 열어둔 상태에서 Calendar 앱까지 확인하기 번거롭다"** — 알림 채널 분산
3. **"일정 전에 부담감이 크다"** — 회의 전 심리적 지원 부재
4. **"AI 캘린더는 비싸다"** — Motion 등 $34/월 vs Cheerly 무료(오픈소스 예정)

### 4.3 잠재적 성장 전략

| Phase | 전략 |
|-------|------|
| Phase 1 (현재) | MVP: 1인 사용자, Teams 알림, Claude AI |
| Phase 2 | KakaoTalk / Slack 채널 추가 → 국내 직장인 타겟 |
| Phase 3 | 팀 기능(공유 일정, 팀 격려) → 기업 B2B |
| Phase 4 | AI 개인화(피드백 학습), 일정 추천 → SaaS 모델 |

---

## 5. 결론

Cheerly는 **"일정 관리 + 감성 AI 격려 + 업무 채널 통합"** 세 요소를 결합한 **블루오션 포지션**에 있습니다.

- 주류 캘린더(Google, Apple)는 격려 메시지가 없음
- 동기부여 앱(Remindly, Rocky)은 일정 관리가 없음
- 업무 채널(Teams) 직접 연동은 모든 경쟁사에서 미지원

MVP 완성 기준으로 **핵심 차별화 3요소가 모두 구현**되어 있으며, Strategy Pattern 아키텍처로 채널 확장 비용이 매우 낮습니다.

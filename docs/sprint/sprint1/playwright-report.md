## Playwright 자동 실행 준비 방법

환경 문제 해결 후 아래 절차로 자동 E2E 테스트 실행 가능:

```bash
# 1. Playwright 브라우저 설치
npx playwright install chromium

# 2. 개발 서버 실행 (별도 터미널)
npm run dev

# 3. E2E 테스트 실행
npx playwright test e2e/

# 4. 결과 보고서 확인
npx playwright show-report
```

현재 `e2e/schedule.spec.ts` 파일이 존재하지만 Vitest에서 실행 시 Playwright 충돌로 인해 실패합니다. `vitest.config.ts`의 `exclude` 설정에 `e2e/**` 추가 권장.
